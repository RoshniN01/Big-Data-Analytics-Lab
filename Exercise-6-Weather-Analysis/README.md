# Exercise 6 - Weather Data Analysis using Hadoop Streaming

## Objective
To analyze weather data using Hadoop Streaming with Python Mapper and Reducer programs.

## Dataset
Weather dataset containing:
- Date
- City
- Maximum Temperature
- Minimum Temperature

## Files
- weather_data.txt
- weather_large.txt
- mapper_max_temp.py
- reducer_max_temp.py
- mapper_min_temp.py
- reducer_min_temp.py

## Operations Performed
- Created weather dataset
- Uploaded dataset to HDFS
- Implemented Python mapper and reducer scripts
- Executed Hadoop Streaming jobs
- Identified the hottest and coldest days from the dataset

## Technologies Used
- Apache Hadoop 3.4.1
- HDFS
- Hadoop Streaming
- Python
- Ubuntu (WSL)

## Outcome
Successfully processed weather data using Hadoop Streaming and generated the highest and lowest temperature records from the dataset.
