# Exercise 1 - Hadoop Installation

## Objective
Install and configure Apache Hadoop in Ubuntu (WSL).

## Software Used
- Ubuntu (WSL)
- Java 17
- Hadoop 3.4.1

## Tasks Performed
- Installed Java
- Installed Hadoop
- Configured JAVA_HOME
- Configured Hadoop environment variables
- Started HDFS services
- Verified installation using JPS

## Output
Successfully installed and configured Hadoop.
